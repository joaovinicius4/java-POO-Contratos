Projeto em Java focado em Programação Orientada a Objetos (POO), utilizando conceitos de:

Composição (Trabalhador possui Contratos e Departamento)

Enums (nível do trabalhador)

Listas (ArrayList para armazenar contratos)

Manipulação de datas com LocalDate e DateTimeFormatter

O programa permite cadastrar trabalhadores, adicionar contratos por hora e calcular a renda mensal de cada trabalhador.
